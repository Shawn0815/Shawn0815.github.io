# 題目描述

給訂一個 array 跟一個 target，找出兩個數字加起來為 target，回傳其 Index。

![image.png](assets/img/2026-04-24-1_Two_Sum.png)

---

# 方法 1: Brute Force

雙層回圈，暴力解。

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        for (int i = 0; i < nums.length; i++) {
            for (int j = i + 1; j < nums.length; j++) {
                if (nums[i] + nums[j] == target) {
                    return new int[]{i, j};
                }
            }
        }

        return new int[]{};
    }
}
```

## 複雜度分析

- Time complexity: $O(n^2)$
    - 雙層迴圈
- Space complexity: $O(1)$
    - 沒使用額外記憶體空間

# 方法 2: Sorting + Two pointer

## 解法

先將陣列排序，用兩個指標分別指向頭尾：

- 若加總 < target，代表總和太小，頭指標右移。
- 若加總 > target，代表總和太小，頭指標右移。

<aside>
💡

**重點**

回傳的陣列要記得排序，小的 Index 要擺在前面，使用 Math.min/max。

</aside>

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int[][] arr = new int[nums.length][2];

        for (int i = 0; i < nums.length; i++) {
            arr[i][0] = nums[i];
            arr[i][1] = i;
        }

        Arrays.sort(arr, (a, b) -> a[0] - b[0]);

        int i = 0, j = nums.length - 1;

        while (i < j) {
            if (arr[i][0] + arr[j][0] == target) {
                return new int[]{Math.min(arr[i][1], arr[j][1]),
                                Math.max(arr[i][1], arr[j][1])};
            }

            if (arr[i][0] + arr[j][0] > target) {
                j--;
            }
            else {
                i++;
            }
        }

        return new int[]{};
    }
}

```

## 複雜度分析

- Time complexity: $O(nlog n)$
    - 排序複雜度：O(n logn)
- Space complexity: $O(n)$
    - 需額外使用大小為 2n 的二維陣列
    - 排序使用 Quick Sort 原地排序，不需額外記憶體

# 方法 3: HashMap (Two pass)

## 解法

先把所有元素加入 set 中，再一個一個 loop 陣列元素，尋找他的夥伴 = target - num，是否存在於 set 中，若存在，則回傳答案。

<aside>
💡

**重點：**

找到夥伴時記得檢查是否等於自己。

</aside>

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            map.put(nums[i], i);
        }

        for (int i = 0; i < nums.length; i++) {
            int diff = target - nums[i];

            if (map.containsKey(diff) && map.get(diff) != i) {
                return new int[]{i, map.get(diff)};
            }
        }

        return new int[]{};
    }
}
```

## 複雜度分析

- Time complexity: $O(n)$
    - 兩個迴圈，分別複雜度 O(n)，總共為 O(2n)
- Space complexity: $O(n)$
    - map 需要存放最多 n 個 元素。

# 方法 4: HashMap (One pass)

## 解法

解法 3 的 one pass 版本，放進 set 前先找有沒有自己的夥伴，沒有才放入。

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();

        // Loop nums array, find partner of each number
        for (int i = 0; i < nums.length; i++) {
            int diff = target - nums[i];

            // If partner exist in map, return answer
            if (map.containsKey(diff)) {
                return new int[] {map.get(diff), i};
            }
            // else, put number into map
            else {
                map.put(nums[i], i);
            }
        }

        return new int[]{};
    }
}
```

## 複雜度分析

- Time complexity: $O(n)$
    - 只走一次迴圈
    - HashMap 的插入與查找複雜度都是 O(1)
- Space complexity: $O(n)$
    - 最壞情況下 HashMap 要存放 n 個元素
